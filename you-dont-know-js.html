Intro to Programming

Code

Often refered to as source code, is a set of special instructions to tell the computer what tasks to perforrm. The rules for valid format are called a computer language or syntax.

Statements

a group of words, numbers and operators that perform a specific task is called a stemenet.

Statement Example

In the following statement:

1b = a * 2;
  • a and b are variables
  • * and = are operators
  • ; indicates the statement is complete.

Therefore the statement is telling the computer to grab the data that is stored in the varible a, times it by two and then store that number as b.

Expressions

Statments are made up of one or more expressions. An Expression is any refrence to a variable or value.

Expression Example

Looking again at our previous statement:

1b = a * 2

There are 4 expressions in this statement:

  • Literal Expression
    2
  • Variable Expression
    a
  • Arithmetic Expression
    a * 2
  • Assignment Expression
    b = a * 2

A final expression type can be explained with this statement:

1alert(a)

These types of expressions are called expression statements.

Executing a program

The program is finally run by compiling the code into something that the computer can understand. This is done through compilers. Different languages approach how and when to compile differently but javascript compiles the program on the fly and then imediatly runs the compiled code.

Output

When executing a program it is very likely that it will output something. Looking at this snippet of code:

123a=2
b= a * 2
console.log(b)
Extra details about line 3
  • log(b) is refered to as a function call where log is being called with the variable b
  • console. is refered to as a object reference.
